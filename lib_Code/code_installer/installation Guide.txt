üì• To Install on Windows (via .bat file):

1. Right-click `bat ) install-code.bat` and choose **"Run as administrator"**
2. A Command Prompt window will open
3. It will install `code.h` globally and show all progress messages.

üßπ To Uninstall on Windows:

    - Show output messages in Command Prompt
